# Accounting Cycle Practice – Green River Computer Consultants (ACR4.4)

Web app for students to practice the full accounting cycle. Enter journal entries using 3-digit codes, view trial balances and ledger.

## Access
- **Repo:** [github.com/jinhanpae/AccCycle_Green-River](https://github.com/jinhanpae/AccCycle_Green-River)
- **Live:** [jinhanpae.github.io/AccCycle_Green-River](https://jinhanpae.github.io/AccCycle_Green-River/)

## Features
- **3 sets**: Transactions (T), Adjustments (A), Closing (C)
- **Exercise mode**: Instant feedback + score tracking
- **Test mode**: No feedback (for quizzes)
- Trial balances, general ledger, chart of accounts popup

## Customize
Edit these files:
- `ACConfig.js` - title, mode, password
- `ACChartOfAccounts.js` - accounts
- `ACQuestions.js` - transactions

## Classroom Notes
- Browser-based, no install needed
- Allow popups for ledger/TB
- Login: name + class password
